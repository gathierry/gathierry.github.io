# Univariable linear regression
- hypothesis function

$$ 
h_{\theta}(x)=\theta_0 + \theta_1 x 
$$

- cost function

$$ J(\theta_0, \theta_1)=\frac{1}{2m}\sum_{i=1}^{m}(h\_{\theta}(x^{(i)})-y^{(i)}))^2 $$

- gradient descend

$$ \theta_j:=\theta_j-\alpha\frac{\partial J}{\partial \theta_j}$$
